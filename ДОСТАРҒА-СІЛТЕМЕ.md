# Достарға Beka AI сілтемесін жіберу

## Маңызды

`http://localhost:5006` — **тек сіздің ноутбукте** жұмыс істейді.  
Достарға осы сілтемені жіберсеңіз — **оларда ашылмайды**.

Интернеттен барлығына ашу үшін **хостинг** керек.

---

## 1-әдіс: Уақытша сілтеме (ngrok) — тез тест

### Орнату
1. https://ngrok.com — тіркелу, ngrok жүктеу
2. `start-beka.bat` іске қосылған болсын (сервер 5006)

### Іске қосу
```cmd
ngrok http 5006
```

### Нәтиже
`https://xxxx.ngrok-free.app` сияқты сілтеме шығады — **осыны достарға жіберіңіз**.

**Ескерту:** ngrok терезесін жапсаңыз — сілтеме өледі. Тегін жоспарда сілтеме әр іске қосқанда өзгеруі мүмкін.

---

## 2-әдіс: Тұрақты hosting (ұзақ мерзім)

| Платформа | Не қояды |
|-----------|----------|
| [Render.com](https://render.com) | Backend + MongoDB Atlas |
| [Railway.app](https://railway.app) | Backend |
| [Vercel](https://vercel.com) | Frontend |
| MongoDB Atlas | Тегін база (бұлт) |

### Не өзгереді
- `MONGO_URI` → MongoDB Atlas сілтемесі
- `GEMINI_API_KEY` → hosting Environment Variables
- Frontend → Vercel, Backend → Render

Бұл жобаны интернетке шығару — жеке қадамдар керек (келесі сұрағанда көмектесуге болады).

---

## 3-әдіс: Бір Wi‑Fi желісінде

Достар **сізбен бір Wi‑Fi**-де болса:

1. `Win + R` → `cmd` → `ipconfig`
2. **IPv4** (мысалы `192.168.1.105`)
3. Досқа: `http://192.168.1.105:5006`
4. Windows Firewall-да 5006 портына рұқсат беру керек

---

## Қорытынды

| Сілтеме | Кімге |
|---------|-------|
| `localhost:5006` | Тек сіз |
| `ngrok https://...` | Интернет (уақытша) |
| Render/Vercel | Барлығына (тұрақты) |
| `192.168.x.x:5006` | Үй Wi‑Fi достары |
