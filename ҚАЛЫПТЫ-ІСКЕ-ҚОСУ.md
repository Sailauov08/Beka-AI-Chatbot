# Beka AI — сайт жабылмай тұруы үшін нұсқау

## Неге жабылады?

`localhost:5174` — бұл **уақытша dev сервер**. Ол тек терминал/`.bat` терезесі **ашық тұрғанда** жұмыс істейді.

| Себеп | Шешім |
|--------|--------|
| Терминал жабылды | Терезені жаппаңыз |
| Cursor қайта ашылды | `.bat` файлдарын қолданыңыз |
| Тек frontend іске қосылды | Backend те қажет |
| OneDrive синхрондау | Жобаны `C:\Projects\` сияқты жерге көшіріңіз |

---

## ⭐ Ең оңай: Бір сервер (ұсынылады)

Бір терезе, бір сілтеме: **http://localhost:5006**

### 1-бет: Frontend құрастыру (бір рет)

`build-frontend.bat` — екі рет шертіңіз.

### 2-бет: Серверді іске қосу

`start-beka.bat` — екі рет шертіңіз. **Терезені жаппаңыз!**

Браузер: **http://localhost:5006**

---

## Ноутбук қосылғанда өзі ашылу (еге шертпейсіз)

**Бір рет орындаңыз:**

1. `build-frontend.bat` (егер әлі істемеген болсаңыз)
2. `install-windows-startup.bat` — екі рет шертіңіз

**Содан кейін:** ноутты сөндіріп қоссаңыз → сервер + браузер **өздері** ашылады.

Сілтеме: **http://localhost:5006**

### MongoDB (маңызды!)

Деректер базасы да автоматты болуы керек:

1. `Win + R` → `services.msc` → Enter  
2. **MongoDB** табыңыз → оң жақ → **Properties**  
3. **Startup type:** `Automatic` → **OK**

Немесе әр қосылғанда MongoDB Compass-ты қолмен ашыңыз.

### Автозапускты өшіру

`Win + R` → `shell:startup` → `Beka-AI-Chatbot` жойыңыз

---

## Dev режим (екі терезе)

Кодты өзгерткенде:

- `start-all-dev.bat` — backend + frontend  
- Браузер: http://localhost:5174

## PM2 (құласа қайта қосады)

```cmd
pm2 start ecosystem.config.cjs
pm2 save
```

Сайт: http://localhost:5006
