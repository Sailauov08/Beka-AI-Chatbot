# Beka AI — нөлден интернетке шығару (толық нұсқау)

Досыңызға жіберетін сілтеме мысалы:
```
https://beka-ai-chatbot.onrender.com
```

---

## 0. Не түсіну керек (бастапқы)

| Сілтеме | Кім көре алады |
|---------|----------------|
| `http://localhost:5006` | **Тек сіздің ноутбук** |
| `https://....onrender.com` | **Барлық әлем** (достар, телефон) |

**localhost** = үйдегі компьютер. Досқа жіберуге **болмайды**.

Интернетке шығару үшін 3 тегін қызмет:
1. **MongoDB Atlas** — деректер базасы (тіркелу, чаттар)
2. **GitHub** — кодты сақтау
3. **Render** — сайтты интернетке қою (HTTPS сілтеме)

Уақыт: шамамен **1–2 сағат** (бірінші рет).

## A1. Жоба дұрыс жұмыс істей ме?

3. Кілтті блокнотқа көшіріңіз (`AIza...` немесе `AQ....`)

Бұл кілтті кейін **Render**-ге қоясыз (GitHub-қа ЕМЕС).

## A3. Git орнату (бір рет)

1. https://git-scm.com/download/win — жүктеп орнатыңыз  
2. PowerShell ашып тексеріңіз:
   ```powershell
   git --version
   ```
   Нұсқа шықса — дұрыс.

# БӨЛІМ B: MongoDB Atlas (бұлттағы база)

## B1. Тіркелу

1. https://www.mongodb.com/cloud/atlas/register  
2. Email / Google арқылы тіркеліңіз  

## B2. Тегін кластер жасау

1. **Build a Database** (немесе **Create**)  
2. **M0 FREE** таңдаңыз → **Create**  
3. Provider: **AWS**, Region: ең жақын (мыс. Frankfurt) → **Create Cluster**  
4. 1–3 минут күтіңіз  

## B3. Пайдаланушы (логин/пароль)

1. Сол жақ **Database Access** (Security)  
2. **Add New Database User**  
3. Authentication: **Password**  
4. Username: `bekauser` (кез келген)  
5. Password: **Generate** → парольді **блокнотқа жазып алыңыз**  
6. Database User Privileges: **Read and write to any database**  
7. **Add User**  

## B4. Желілік рұқсат (маңызды!)

1. **Network Access**  
2. **Add IP Address**  
3. **Allow Access from Anywhere** → `0.0.0.0/0`  
4. **Confirm**  

(Бұл hosting-тен базаға қосылу үшін керек.)

## B5. Connection string (сілтеме)

1. **Database** → **Connect**  
2. **Drivers**  
3. **Node.js**  
4. Connection string көшіріңіз, мысал:
   ```
   mongodb+srv://bekauser:<password>@cluster0.xxxxx.mongodb.net/?retryWrites=true&w=majority
   ```
5. `<password>` орнына **B3-тегі нақты парольді** қойыңыз (жақша <> жойыңыз)  
6. Соңына база атын қосыңыз — `.net/` кейін:
   ```
   mongodb+srv://bekauser:ПАРОЛЬ@cluster0.xxxxx.mongodb.net/ai_chatbot_db?retryWrites=true&w=majority
   ```

Осы жолды **MONGO_URI** деп атаймыз — блокнотта сақтаңыз.

# БӨЛІМ C: GitHub (код жүктеу)

## C1. GitHub тіркелу

https://github.com/signup

## C2. Жаңа репозиторий

1. GitHub → оң жақ **+** → **New repository**  
2. Name: `Beka-AI-Chatbot`  
3. **Public**  
4. README, .gitignore қоспаңыз (бос қалдырыңыз)  
5. **Create repository**  

## C3. Кодты жіберу (PowerShell)

```powershell
cd "C:\Users\USER\OneDrive\Desktop\Beka-AI-Chatbot"

git init
git add .
git commit -m "Beka AI Chatbot first deploy"
```

GitHub бетінде көрсетілген сілтемені қолданыңыз (`СІЗДІҢ_ЛОГИН` ауыстырыңыз):

```powershell
git remote add origin https://github.com/СІЗДІҢ_ЛОГИН/Beka-AI-Chatbot.git
git branch -M main
git push -u origin main
```

GitHub логин/пароль сұраса — кіріңіз (немесе Personal Access Token).

**Тексеру:** GitHub-та файлдар көрінуі керек (`backend`, `frontend`, `package.json`).

**Маңызды:** `backend/.env` жіберілмейді (`.gitignore` қорғайды) — дұрыс!

# БӨЛІМ D: Render (интернет сайты)

## D1. Тіркелу

1. https://render.com  
2. **Get Started** → **Sign up with GitHub**  
3. GitHub рұқсатын беріңіз  

## D2. Web Service жасау

1. Dashboard → **New +** → **Web Service**  
2. **Connect** `Beka-AI-Chatbot` репозиторийі  
3. Баптаулар:

| Өріс | Мән |
|------|-----|
| **Name** | `beka-ai-chatbot` (сілтемеде көрінеді) |
| **Region** | Frankfurt (EU) |
| **Branch** | `main` |
| **Runtime** | `Node` |
| **Build Command** | `npm run build && cd backend && npm install` |
| **Start Command** | `cd backend && npm start` |
| **Instance Type** | **Free** |

## D3. Environment Variables (құпия мәндер)

**Environment** → **Add Environment Variable** — әрқайсысын:

| Key | Value (мысал) |
|-----|----------------|
| `NODE_ENV` | `production` |
| `SERVE_FRONTEND` | `true` |
| `MONGO_URI` | B5-тегі толық Atlas сілтеме |
| `GEMINI_API_KEY` | Google AI Studio кілті |
| `JWT_SECRET` | `BekaSecret2024MyRandomKey99` (өзіңіз ойлап жазыңыз) |
| `GEMINI_MODEL` | `gemini-2.5-flash,gemini-2.0-flash-lite` |

## D4. Deploy

1. **Create Web Service**  
2. **Logs** қараңыз — 5–15 минут  
3. Жасыл **Live** болса — дайын  

## D5. Сілтемені алу

Жоғарыда:
```
https://beka-ai-chatbot.onrender.com
```
(сіздің Name-ге байланысты өзгеруі мүмкін)

Браузерде ашыңыз → тіркелу → чат тесті.
