# Beka AI — төлем Lava.top

Сіз **lava.top**-та тіркелдіңіз — керемет! Төмендегі қадамдарды орындаңыз.

Ақша **lava.top шотыңызға** түседі, содан кейін **картаңызға** шығара аласыз (платформа шарттары бойынша).

---

## Қадам 1: Premium өнім жасау

1. https://app.lava.top (немесе lava.top кабинеті) — кіріңіз
2. **Жаңа өнім** / **Цифровой продукт** жасаңыз
3. Атауы: `Beka AI Premium`
4. **Баға** қойыңыз (мысалы 2990 ₸ немесе 990 ₽ — lava валютасын таңдайсыз)
5. Төлем түрі: lava-да қандай болса — кодқа сәйкес `LAVA_PERIODICITY` қойыңыз (`MONTHLY` немесе `ONE_TIME`)
6. Өнімді **сақтаңыз** және **жариялаңыз**

---

## Қадам 2: API кілт алу

1. Кабинет → **Интеграции** → **Public API**
2. **Создать API key** → кілтті көшіріп сақтаңыз (бір рет көрсетіледі!)
3. Бұл — `LAVA_API_KEY`

---

## Қадам 3: Offer ID табу

Әр өнімнің **offer** (ұсыныс) ID бар — ол кодқа керек.

**Әдіс A — кабинеттен:**  
Өнімді ашып, сілтемеде немесе өнім баптауларында `offer` UUID көруге болады.

**Әдіс B — API арқылы (код бапталғаннан кейін):**  
Сайтқа кіріп, браузерде (әкімші ретінде):
```
GET https://beka-ai-chatbot.onrender.com/api/payment/lava-products
Authorization: Bearer СІЗДІҢ_JWT
```
Жауапта `offerId` көшіріңіз → `LAVA_OFFER_ID`

---

## Қадам 4: Webhook қосу

1. **Интеграции** → API key жанындағы **⋮** → **Добавить Webhook**
2. **URL:**
   ```
   https://beka-ai-chatbot.onrender.com/api/payment/lava/webhook
   ```
   (жергілікті тест: ngrok URL + `/api/payment/lava/webhook`)
3. **Тип события:** `Результат платежа` (payment.success / payment.failed)
4. **Аутентификация:** `API key вашего сервиса`
5. Кілт ойлап табыңыз (мысалы `beka_webhook_secret_2026`) — бұл `LAVA_WEBHOOK_API_KEY`
6. Сақтаңыз

⚠️ Webhook **публикалық** болуы керек — логин қажет емес.

---

## Қадам 5: .env / Render

`backend/.env` немесе **Render → Environment:**

```env
APP_URL=https://beka-ai-chatbot.onrender.com
CLIENT_URL=https://beka-ai-chatbot.onrender.com

LAVA_API_KEY=сіздің_api_key
LAVA_OFFER_ID=сіздің_offer_uuid
LAVA_WEBHOOK_API_KEY=webhook_кілтіңіз
LAVA_CURRENCY=RUB
LAVA_PERIODICITY=MONTHLY

PREMIUM_DURATION_DAYS=30
PREMIUM_DISPLAY_PRICE=990 ₽ / 30 күн

OWNER_EMAIL=сіздің@email.com
```

**Валюта:** lava өніміңіз қандай валютада болса — `RUB`, `USD` немесе `EUR` (KZT тікелей жоқ болса RUB қолданыңыз).
